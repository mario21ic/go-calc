# go-calc
Files to practice golang

## On localhost

Dependencies:
```
make deps
```

Lint:
```
make lint
```

Tests:
```
make tests
```

Build:
```
make build
```

Docker:
```
make docker
```


## Using Docker

Docker Image Build:
```
./docker-image-build.sh
```

Dependencies:
```
./deps-con-docker.sh
```

Lint:
```
./lint-con-docker.sh
```

Tests:
```
./tests-con-docker.sh
```

Build:
```
./build-con-docker.sh
```
